# 서비스 소개
<strong>"호빵"</strong>은 <strong>호구빵명</strong>의 줄임말로, 소비자가 창호의 종류와 크기만 입력하면 합리적인 가격을 쉽게 확인할 수 있도록 돕는 가격 지표 앱입니다. 복잡한 계산 없이도 창호의 적정 가격대를 제시하여, 불필요한 지출을 방지하고 소비자가 안심하고 선택할 수 있게 지원합니다.
